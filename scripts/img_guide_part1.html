BÖLÜM 1

Genel Bakış ve Strateji

2500 tarifin her birine, aynı estetik standartta profesyonel kalitede bir görsel gerekir. Bunları manuel hazırlamak haftalar sürer ve tutarsız sonuç verir. Çözüm: hibrit kaynak stratejisi ile yarı otomatik üretim.

1.1 Hibrit Kaynak Yaklaşımı

Pipeline, her tarif için kaynakları öncelik sırasıyla dener. Ücretsiz stok fotoğraf siteleri önce taranır; yeterli kalitede sonuç bulunamazsa yapay zeka ile görsel üretilir. Bu yaklaşım maliyeti minimumda tutar.

  1. Unsplash — en kaliteli, telifsiz; yaygın yemeklerin ~%45'i bulunur
  2. Pexels — Unsplash bulamazsa devreye girer
  3. Pixabay — geniş havuz, kalite değişken; son ücretsiz şans
  4. DALL·E 3 (AI) — yöresel Türk yemekleri için özel prompt'la üretim
  5. Flux Pro (AI) — alternatif, daha fotogerçekçi AI motoru
  6. Manuel — admin onayı / özel çekim (son çare)
💡 Neden Hibrit?
"Pizza", "salata" gibi yaygın yemekler stok sitelerinde bol bulunur — bunlar ücretsizdir. Ama "perde pilavı", "su böreği" gibi yöresel yemekler nadiren bulunur; bunlar için DALL·E 3, "geleneksel Türk sunumu" yönergesiyle özgün görsel üretir.

Görsel Kaynak Kanalları ve API Anahtarları

Bu bölüm, her kaynağın API anahtarının nereden ve nasıl alınacağını, ücretlendirme ve oran limitlerini açıklar.

2.1 Unsplash API

U
Unsplash
ÜCRETSİZ

En yüksek kaliteli telifsiz fotoğraf kaynağı. Birincil tercih. Atıf (fotoğrafçı adı) gösterimi zorunludur.

API Anahtarı Nasıl Alınır?

  1. unsplash.com/developers adresine gidin, hesap açın.
  2. "New Application" → kullanım koşullarını kabul edin.
  3. Uygulama adı ve açıklaması girin (örn. "Pratik Tarifler — recipe images").
  4. Oluşan ekranda Access Key'i kopyalayın.
  5. Başlangıçta "Demo" tier (50 istek/saat). Üretim için "Apply for Production" ile başvurun — onaylanınca limit 5.000 istek/saate çıkar.
ÖzellikDeğer
Endpointhttps://api.unsplash.com/search/photos
Kimlik doğrulamaAuthorization: Client-ID {ACCESS_KEY}
Demo limiti50 istek/saat
Üretim limiti5.000 istek/saat
MaliyetÜcretsiz
Env değişkeniUNSPLASH_ACCESS_KEY
⚠️ Unsplash Atıf Kuralı
Unsplash, indirme işleminde download_location endpoint'inin tetiklenmesini şart koşar. Ayrıca fotoğrafçı adı ve profili uygulamada gösterilmelidir. Pipeline kodu bunu otomatik yapar (Bölüm 4.2).

2.2 Pexels API

P
Pexels
ÜCRETSİZ

Unsplash'a güçlü bir alternatif. Atıf zorunlu değildir ama önerilir.

API Anahtarı Nasıl Alınır?

  1. pexels.com/api adresine gidin, "Get Started"a tıklayın.
  2. Hesap oluşturun veya giriş yapın.
  3. Kullanım amacını kısaca yazın (örn. "mobile recipe app").
  4. API anahtarı anında üretilir — panodan kopyalayın.
